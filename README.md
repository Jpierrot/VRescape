# VRescape

### 활용 sdk   
=> picoVR sdk(무료)   
 
### 활용 에셋   
=> Morgue Room PBR(무료)
=> Key and Lock(무료)

### 플레이 방식   
wasd를 활용하여 앞뒤좌우로 움직입니다.

마우스 움직임을 통해 시야를 전환할 수 있습니다.

마우스 클릭을 통해 오브젝트와 상호작용 할 수 있습니다.

열쇠를 찾아서 열쇠로 문을 열고 탈출하면 됩니다.
